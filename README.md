# ExBanking

OTP_banking: a pure elixir application that stores state in memory using GenServer

how to run:

1) clone this repo
2) cd OTP_Banking
3) at the project root, run  `mix deps.get`
4) start application with: `iex -S mix`
5) run `h ExBanking` for usage example
